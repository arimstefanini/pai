# ✅ CHECKLIST – VALIDAÇÃO DO SITE

Antes de qualquer mudança ou deploy, validar:

---

## 🧠 ESTRATÉGIA

* [ ] O site ainda foca em customização?
* [ ] O fluxo principal está preservado?

---

## 🔥 HERO

* [ ] Comunica em 3 segundos?
* [ ] Tem CTA claro?
* [ ] Não mostra preço indevido?

---

## 🎯 FLUXO

* [ ] Usuário chega ao orçamento em até 2 cliques?
* [ ] Existe caminho claro para ação?

---

## 🧲 CTA

* [ ] Todos os blocos têm CTA?
* [ ] Botões são visíveis?

---

## 🔁 CUSTOMIZAÇÃO

* [ ] Todos os produtos têm "Quero algo parecido"?
* [ ] O formulário recebe contexto?

---

## 📱 MOBILE

* [ ] Funciona perfeitamente no celular?
* [ ] Botão fixo está visível?

---

## ⚙️ TÉCNICO

* [ ] Código limpo?
* [ ] Sem duplicação?
* [ ] Dados organizados?

---

## 💬 COPY

* [ ] Linguagem natural?
* [ ] Sem texto forçado?
* [ ] Clareza?

---

## 🚀 CONVERSÃO

* [ ] O usuário sabe o que fazer?
* [ ] Existe fricção desnecessária?
* [ ] Está fácil agir?

---

## ❗ REGRA FINAL

Se você não compraria → o usuário também não
