# 🎨 UI & UX RULES – 3D CUSTOM PLATFORM

## 🎯 Objetivo

Criar uma interface limpa, intuitiva e altamente conversiva.

---

## 🧠 PRINCÍPIO CENTRAL

Menos fricção = mais conversão

---

## 🏠 HERO

* Deve comunicar valor em 3 segundos
* Foco em customização (não produto)
* CTA claro:

  * Criar minha ideia
  * Ver exemplos

---

## 🔘 BOTÕES (CTA)

* Sempre visíveis

* Texto claro e direto:

  * "Criar minha ideia"
  * "Quero algo parecido"
  * "Comprar agora"

* Cor contrastante

* Hover com feedback

---

## 📐 LAYOUT

* Muito espaço em branco
* Grid simples
* Evitar poluição visual
* Mobile-first sempre

---

## 🧩 HIERARQUIA

Ordem de atenção:

1. Headline
2. Imagem
3. CTA
4. Texto secundário

---

## 🖼️ IMAGENS

* Devem parecer premium
* Fundo limpo
* Iluminação suave
* Mostrar contexto real (ambiente)

---

## 🧠 DECISÃO DO USUÁRIO

Sempre facilitar:

* mostrar exemplos
* reduzir dúvida
* dar caminho claro

---

## 🔁 CUSTOMIZAÇÃO (CRÍTICO)

Todo item deve ter:

→ "Quero algo parecido"

---

## 📱 MOBILE

* Botão fixo na base
* Texto legível
* Espaçamento confortável

---

## ⚠️ EVITAR

* excesso de texto
* múltiplas ações confusas
* telas sem CTA
* layouts complexos

---

## ❗ REGRA DE OURO

Se o usuário precisa pensar demais → o design falhou
