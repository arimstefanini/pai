## 🚨 REGRAS CRÍTICAS

1. Nunca criar UI diretamente na página  
   → sempre usar componentes reutilizáveis

2. Sempre priorizar:

- conversão
- baixa fricção
- clareza

3. Sempre manter:

- no máximo 2 cliques até ação
- CTA visível

4. Sempre usar design system  
   → não usar estilos soltos

5. Todo produto deve permitir:  
   → "Quero algo parecido"
